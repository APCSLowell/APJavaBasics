AP Java: a Basic Java Program

You've written programs before in some language like JavaScript or Python. In many ways Java programming is similar, but you will notice immediately that the basic Java "Hello, World!" program looks pretty confusing and complicated. Here's a Java program that prints "Hello, World!" to the console. Press the Run button and the program output should appear to the right of the code in what is called the console.

Luckily though, one line of code is much more important and interesting than all the rest. For now we are just interested in

System.out.println("Hello, World!");

and we will ignore everything else. This line of code an example of a Java statement. If we think of Java as a language like English, a statement is like a sentence. The same way a period ends an English sentence, a semi-colon marks the end of of a Java statement.

"Hello World!" is an example of a Java String. A String is a collection of letters, digits, punctuation and/or spaces. The beginning and end of the String are marked with double quotes (").
